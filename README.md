# Trabajo Práctico N°3 - JavaScript

Este repositorio contiene la resolución del Trabajo Práctico N°3 de la materia **Programación**, correspondiente a la Tecnicatura en Análisis de Sistemas Informáticos.

## Archivos

- `index.html`: Documento principal que vincula ambos scripts y permite probar la funcionalidad del DOM directamente en el navegador.
- `DOM.js`: Contiene los ejercicios 9, 10 y 11, enfocados en la manipulación del DOM con JavaScript.
- `objetos.js`: Contiene los ejercicios 6, 7 y 8, desarrollados utilizando clases y objetos en JavaScript.
